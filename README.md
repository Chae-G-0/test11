### 기능

-   useState, useEffect 이용해 상태값 관리
-   router를 이용해 SPA 구현
-   axios를 이용해 공공데이터 기상청 API를 받아와 화면에 기상 정보 안내
-   react slick을 이용해 다양한 종류의 슬라이드 구현
-   scolltop 값을 이용해 이벤트를 붙여 스크롤에 따라 콘텐츠가 나타나는 효과 구현
